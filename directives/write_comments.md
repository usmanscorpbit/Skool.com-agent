# Write Comments & Replies

## Goal
Generate 5 thoughtful comment options for a Skool post, and craft replies to existing comments.

## Inputs
- **post_url** (required): URL of the post to comment on
- **post_content** (optional): Copy-pasted post content if URL not accessible
- **existing_comments** (optional): Comments to reply to
- **your_expertise** (optional): Your background for credibility

## Workflow

### For New Comments

#### Step 1: Analyze the Post
- Read the post title and content
- Identify the main topic/question
- Note the tone (casual, professional, vulnerable, educational)

#### Step 2: Generate 5 Comment Options

**Type A: Add Value**
```
[Relate to their point] + [Add new insight/tip] + [Question back]
```

**Type B: Share Experience**
```
[Personal story that relates] + [What you learned] + [Encouragement]
```

**Type C: Ask Deeper Question**
```
[Acknowledge their point] + [Thoughtful follow-up question]
```

**Type D: Respectful Challenge**
```
[Appreciate their perspective] + [Offer alternative view] + [Invite discussion]
```

**Type E: Resource/Help Offer**
```
[Validate their situation] + [Offer specific help or resource]
```

### For Replies to Comments

#### Step 1: Read the Comment Thread
- Understand the original comment's intent
- Note any questions asked
- Identify opportunity to add value

#### Step 2: Generate Reply
- Acknowledge their point
- Add value (don't just agree)
- Keep it conversational
- End with engagement hook if natural

## Output Format

```
=== 5 COMMENT OPTIONS ===

1. [VALUE-ADD]
[Comment text]

2. [EXPERIENCE SHARE]
[Comment text]

3. [DEEPER QUESTION]
[Comment text]

4. [ALTERNATIVE VIEW]
[Comment text]

5. [HELP OFFER]
[Comment text]

---
RECOMMENDATION: Option [X] because [reason]

=== REPLIES TO EXISTING COMMENTS ===

Reply to [Author]:
[Reply text]
```

## Comment Best Practices

### DO:
- Read the full post before commenting
- Add genuine value, not just "Great post!"
- Share relevant personal experience
- Ask thoughtful questions
- Be specific, not generic
- Match the energy/tone of the post

### DON'T:
- Self-promote in first comment
- Write walls of text
- Be contrarian for attention
- Use generic praise ("Love this!")
- Hijack the conversation
- Comment without reading

## Engagement Strategy

**For visibility:**
- Comment early (within first few hours)
- Reply to other comments to stay in notifications
- Ask questions that invite the OP to respond

**For relationship building:**
- Reference something specific from their post
- Share a genuine personal connection
- Offer to help with something specific
